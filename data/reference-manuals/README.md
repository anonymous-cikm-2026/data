# Reference Manuals

This directory contains the public manuals used in the paper.

Included manual sets:

- `ICD-addendums/`: 2018-2020 ICD-10-CM addenda text files used to exclude changed codes from the strict 2017-2019 MIMIC build.
- `ICD-2019-manual/`: ICD-10-CM 2019 coding guidelines and tabular/index source files.
- `legal/usc_title18/`: Title 18 HTML snapshots used for legal retrieval.
- `legal/ussg/`: United States Sentencing Guidelines manuals used for sentencing retrieval.

Anonymity guardrails:

- only public source manuals are committed here,
- extracted text exports, download manifests, and logs are intentionally excluded,
- no files in this directory should reveal local machine paths or source-project workflow details.
